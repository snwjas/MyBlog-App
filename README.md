# MyBlog App

MyBlog前台台前端项目。



以前的课后作业改的，写的很烂，凑合能用。



后台项目管理：[Github](https://github.com/snwjas/MyBlog) | [Gitee](https://gitee.com/snwjas/my-blog)



## 快速开始

安装依赖：

```shell
npm install
```

开发环境运行：

```shell
npm run dev
```

静态资源打包：

```shell
npm run build
```



修改基本应用请求API

修改根目录下的`.env.production`（生产环境）和`.env.development`（开发环境）文件下的`VUE_APP_BASE_API`
